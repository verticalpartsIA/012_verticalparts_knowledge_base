# Testes da Factory

## Objetivo

Definir a estrutura inicial de testes para a Omie Knowledge Factory.

## Categorias planejadas

- Testes de parser com fixtures locais.
- Testes de extração de métodos.
- Testes de renderização de templates.
- Testes de deduplicação de perguntas.
- Testes de chunking.
- Testes de quality gate.
- Testes de ausência de segredos.
