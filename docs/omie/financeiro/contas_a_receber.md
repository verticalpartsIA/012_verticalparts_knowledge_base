# Contas a Receber

## Título

Contas a receber

## Domínio

Omie Financeiro

## Endpoint

`/financas/contareceber/`

## Métodos conhecidos

- `ListarContasReceber`
- `ConsultarContaReceber`
- `IncluirContaReceber`
- `AlterarContaReceber`
- `ExcluirContaReceber`

## Quando usar

Use este endpoint para consultar e manter títulos a receber de clientes, parcelas comerciais, cobranças e registros financeiros vinculados a vendas ou serviços.

## Entidades relacionadas

- Cliente
- Pedido de venda
- Ordem de serviço
- Categoria financeira
- Movimento financeiro
- Documento fiscal

## Exemplos de perguntas que um usuário faria

- Como listar contas a receber em aberto?
- Como consultar uma parcela de cliente?
- Qual endpoint registra um título a receber?
- Como relacionar pedido de venda e conta a receber?

## Observações para RAG

Recupere este documento para dúvidas sobre recebíveis, cobranças, parcelas, clientes devedores, vencimentos e conciliação de receitas. Não assumir fluxo de cobrança sem validação documental.
